# Suhaib Premium Portfolio

Modern responsive portfolio website built with Next.js, TypeScript and Tailwind CSS v4.

## Run locally

```bash
npm install
npm run dev
```

Open:

```bash
http://localhost:3000
```

## Important

Edit your real GitHub and LinkedIn links in:

```bash
data/site.ts
```

## Deploy

Push to GitHub. Vercel will deploy automatically.
